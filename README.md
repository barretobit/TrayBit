# TrayBit

A low-latency hardware telemetry widget for the Windows 11 taskbar. Displays real-time system metrics directly in the taskbar area.

## Features

- **CPU** — real-time usage percentage and per-core stats
- **RAM** — memory usage and available memory
- **CPU Temperatures** — per-core thermal readings
- **GPU** — utilization, temperature, and memory usage
- **Network** — upload/download throughput
- **Battery** — charge percentage and power status

Designed for power users who want glanceable system performance data without opening Task Manager or third-party overlays.

## Target

Personal tool — built for my own Windows 11 machine.

## Tech Stack

- .NET 10 + WPF
- Windows 11 taskbar integration
